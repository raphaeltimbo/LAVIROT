Disk Element¶

class LaviRot.elements.DiskElement(n, material, width, i_d, o_d)[source]¶

A disk element.

This class will create a disk element.

Parameters:

n: int

Node in which the disk will be inserted.

material : LaviRot.Material

Shaft material.

width: float

The disk width.

i_d: float

Inner diameter.

o_d: float

Outer diameter.

References

[R1]‘Dynamics of Rotating Machinery’ by MI Friswell, JET Penny, SD Garvey & AW Lees, published by Cambridge University Press, 2010 pp. 156-157.

Examples

>>> from LaviRot.materials import steel
>>> disk = DiskElement(0, steel, 0.07, 0.05, 0.28)
>>> disk.Ip
0.32956362089137037

Attributes

m (float) Mass of the disk element.
Id (float) Diametral moment of inertia.
Ip (float) Polar moment of inertia
