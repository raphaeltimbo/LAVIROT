Rotor¶

class LaviRot.rotor.Rotor(shaft_elements, disk_elements, bearing_seal_elements, w=0)[source]¶

A rotor object.

This class will create a rotor with the shaft, disk and bearing elements provided.

Parameters:

shaft_elements: list

List with the shaft elements

disk_elements: list

List with the disk elements

bearing_elements: list

List with the bearing elements

w: float, optional

Rotor speed. Defaults to 0.
Returns:

A rotor object.

Examples

>>> #  Rotor without damping with 2 shaft elements 1 disk and 2 bearings
>>> n = 1
>>> z = 0
>>> le = 0.25
>>> i_d = 0
>>> o_d = 0.05
>>> E = 211e9
>>> G = 81.2e9
>>> rho = 7810
>>> tim0 = ShaftElement(0, le, i_d, o_d, E, G, rho,
...                    shear_effects=True,
...                    rotary_inertia=True,
...                    gyroscopic=True)
>>> tim1 = ShaftElement(1, le, i_d, o_d, E, G, rho,
...                    shear_effects=True,
...                    rotary_inertia=True,
...                    gyroscopic=True)
>>> shaft_elm = [tim0, tim1]
>>> disk0 = DiskElement(1, rho, 0.07, 0.05, 0.28)
>>> stf = 1e6
>>> bearing0 = BearingElement(0, kxx=stf, cxx=0)
>>> bearing1 = BearingElement(2, kxx=stf, cxx=0)
>>> rotor = Rotor(shaft_elm, [disk0], [bearing0, bearing1])
>>> rotor.wd[0] 
34.2773...
